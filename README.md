# SMG_Map_Visualization
서울시 열린데이터광장(https://data.seoul.go.kr/)에서 제공하는 '서울시 조직도 및 직원업무 정보'에서 자료를 받아 데이터 전처리를 진행한 후에 카카오맵API를 활용하여 위도/경도를 추출. 이후 완성된 dataframe으로 map visualization

## 사용 라이브러리
* pandas
* requests
* folium
